# Paper Reading Skills

Public skill pack containing:

- `paper-deep-read`: deep reading workflow for a single research paper.
- `paper-review`: comparative literature review workflow across multiple papers.

## Structure

- `skills/paper-deep-read/SKILL.md`
- `skills/paper-review/SKILL.md`

## Usage

Copy the skill folders into your Codex/Agents skills directory, or reference these `SKILL.md` files directly.
